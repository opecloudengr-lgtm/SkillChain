Task:

Generate the next smart contract module for SkillChain.

Requirements:

1. Explain the contract purpose.
2. List storage variables.
3. List events.
4. List custom errors.
5. Explain access control.
6. Generate the Solidity contract.
7. Generate deployment script updates.
8. Generate Hardhat tests.
9. Explain how the frontend will interact with it.

Use OpenZeppelin.
Use Solidity ^0.8.x.
Do not use upgradeable contracts unless requested.
